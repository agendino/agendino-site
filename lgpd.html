Conformidade LGPD

O Agendino está totalmente comprometido com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018). Esta página detalha como garantimos a conformidade e protegemos os direitos dos titulares de dados.

1. O Agendino como Operador

Na maioria das situações, o Agendino atua como Operador de dados pessoais, processando informações em nome de nossos clientes (os Controladores), que utilizam o sistema para gerenciar seus agendamentos.

2. Direitos do Titular

Garantimos ferramentas para que os titulares de dados possam exercer seus direitos, incluindo:

  • Confirmação da existência de tratamento.
  • Acesso aos dados.
  • Correção de dados incompletos ou inexatos.
  • Eliminação de dados pessoais (quando aplicável).
  • Portabilidade dos dados.

3. Medidas de Segurança

Implementamos medidas técnicas e administrativas para proteger os dados pessoais de acessos não autorizados e de situações acidentais ou ilícitas de destruição, perda, alteração ou comunicação.

4. Armazenamento de Dados

Os dados são armazenados em servidores de alta segurança, com backups regulares e monitoramento constante contra vulnerabilidades.

5. Encarregado de Dados (DPO)

Para questões relacionadas à LGPD, você pode entrar em contato com nosso Encarregado de Proteção de Dados através do e-mail: dpo@agendino.com.br
